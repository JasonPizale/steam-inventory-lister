# Steam Inventory Lister
A utility that analyzes your Steam inventory, fetches market prices, and helps you prepare multiple items for sale more efficiently.

# Project Goals
- make it easier to handle large steam inventories
- fetch and display item values
- help prepare bulk listings without violating Steam's restrictions
- improve the speed and reduce repetitive manual actions

# Features (Planned)
- Fetch Steam inventory data
- Pull market prices for each item
- Filter items
- Assist with opening listing pages quickly

# Tech Stack
- Python 3.x
- Requests of HTTP client library
- (Optional) Async for faster price fetching
- (Optional) Simple UI or CLI
